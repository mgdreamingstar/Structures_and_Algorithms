## 检查链表代码是否正确的边界条件

1. 如果链表为空时，代码是否能正常工作
2. 如果链表只包含一个节点时，代码是否能正常工作
3. 如果链表只包含两个节点时，代码是否能正常工作
4. 代码逻辑在处理头节点和尾节点的时候，代码是否能正常工作

## 需要熟练的链表操作

1. 单链表反转
2. 链表中环的检测
3. 两个有序的链表合并
4. 删除链表倒数第 n 个节点
5. 求链表的中间节点
